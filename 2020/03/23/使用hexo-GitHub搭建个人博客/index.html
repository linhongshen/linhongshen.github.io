使用hexo+GitHub搭建个人博客
发表于 2020-03-23| 更新于 2020-03-23
阅读量:

前言

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。静态网页可以一键部署至GitHub或其它静态服务器上，不需要自行购买域名和服务器即可拥有个人博客。当然，如果你拥有域名和服务器也可以很方便的部署。

使用该方法搭建的博客，需要使用markdown语法来编写你的文章。请提前熟悉markdown的语法，可以点击这里去学习。

下面将介绍具体的博客搭建部署过程。

Hexo

Hexo的安装非常简单，通常只需要几分钟就能搞定！

Hexo依赖于以下软件：
|软件|版本|
|-|-|
|Node.js|>10.0|
|Git|最新版|

本次介绍安装的hexo-cli版本为3.1.0，不排除后续版本需要更高版本的nodejs。请参考hexo的安装前提。如果Git和Node.js已经安装了，可以直接跳转至Hexo安装。如果还未安装则跟随后续步骤进行Git和Node.js的安装。

下面将详细介绍Hexo的安装过程。

Git安装

  • 介绍

    Git（读音为/gɪt/）是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。 [1] Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

  • 下载
    打开git官网下载页面，选择电脑的系统，下载安装包。

    安装时不需要选择特殊选项，一直点下一步直到安装完成即可。

  • 安装
